# ClawAgent

A hardened local AI agent runtime for Windows. Single agent,
WSL2 sandbox, egress firewall, loopback-only gateway, DPAPI key
storage, kill switch. One-click installer.

## Security substrate

Same as ClawFactory — WSL2 isolation, nftables egress firewall,
automount=false, loopback-only gateway, Windows Firewall inbound
block, DPAPI key storage, Kill Switch.

## Installation

Download ClawAgent-Setup.exe, run as Administrator.

## Upgrade to ClawFactory

ClawFactory adds 5 agent profiles, factory orchestration scaffolding,
and switch-provider support. Available at https://clawfactory.app
